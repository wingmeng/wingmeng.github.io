基于 CSS + SVG 实现的可灵活配置环形进度条组件，支持基础环形、仪表盘式两种形态。

**技术亮点：**

- 通过 SVG 的 stroke-dasharray 结合弧长公式实现进度可视化
- 依托 CSS 变量实现进度、尺寸、环宽、颜色的动态控制
- 代码高复用、易扩展，可快速集成到各类前端项目中
